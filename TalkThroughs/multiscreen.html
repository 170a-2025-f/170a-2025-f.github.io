ideas, q&a

q: Images work but video breaks everythimg, what's going on?

a: It could be several things, but make sure that you're not putting the video url string inside an array. The array is just for images.

Good: fscreens.push(new Fscreen("movies/1.mp4",true));

Bad: fscreens.push(new Fscreen(["movies/1.mp4"],true));

Q: What if I want different animation timers on each item?

A: Hmm... there at least a few ways to do this. instead of animate() being a function that runs and controls all of the Fscreens together, you could rewrite it to be a method of Fscreen, so that each Fscreen has it as a method. Then you could have a property on the fscreen (set as another parameter to the constructor, also w/a default?). Something like that.

Another approach would be to add a bunch of if statements in animate that makes exceptions based on the id of the fscreen
